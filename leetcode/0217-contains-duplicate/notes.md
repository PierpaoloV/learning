# Notes — Contains Duplicate

## My Approach
I just thought that the most optimal way to solve this problem is to remove the duplicates and then compare the length of the array with the length of the array without duplicates. If the lengths are the same, then there are no duplicates. If the lengths are different, then there are duplicates.

## Complexity
- Time: O(n log n)
- Space: O(n)
