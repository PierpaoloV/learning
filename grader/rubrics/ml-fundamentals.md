# ML Fundamentals Grading Rubric

Grade the answer on these 4 dimensions (1–5 each):

## 1. Conceptual Accuracy (1–5)
- 5: Technically correct, no errors
- 3: Mostly correct with minor inaccuracies
- 1: Fundamental misunderstanding

## 2. Depth (1–5)
- 5: Goes beyond surface definition — explains WHY, covers edge cases
- 3: Correct definition but shallow
- 1: Only surface-level or memorized answer

## 3. Practical Awareness (1–5)
- 5: Knows when to use it, common failure modes, real-world caveats
- 3: Some practical knowledge
- 1: No practical context

## 4. Communication Clarity (1–5)
- 5: Would satisfy a senior ML interviewer at Google/Meta/Anthropic
- 3: Understandable but disorganized
- 1: Unclear or jargon without explanation

## Output format
Return a markdown report with score per dimension, total /20, what was missing,
and a model answer the candidate should study.
